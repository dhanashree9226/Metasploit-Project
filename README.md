# Metasploit-Project
Metasploit Security Assessment Project using Kali Linux, Nmap, Shodan, and Metasploit Framework.

Student Information

Name: Dhanashree Madvan

Project Objective

To perform reconnaissance, service enumeration, vulnerability identification, and Metasploit module analysis in a controlled cybersecurity lab environment.

Tools Used

- Kali Linux
- Nmap
- Shodan
- Metasploit Framework
- VMware Workstation
- Metasploitable 2

Project Activities

1. Shodan Research
2. Target Identification
3. Nmap Port Scanning
4. Service Detection
5. Operating System Detection
6. Vulnerability Analysis
7. Metasploit Module Research
8. Documentation and Reporting

Target Machine

Metasploitable 2

Results

- Identified open ports and services
- Performed service enumeration
- Detected operating system details
- Analyzed potential vulnerabilities
- Researched relevant Metasploit modules
- Created security assessment documentation
